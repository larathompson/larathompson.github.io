Snake

Type: HTML, CSS, JavaScript Game | Duration: 1 week | Project: Independent


For my first project, I designed and created the retro, grid-based game, Snake using HTML, CSS and JavaScript. The aim of the game is for the snake to eat the food which is randomely placed around the board. If successful, the snake grows in size and its speed increases. However, if the snake hits the wall, the user loses a life. In order to make the game more entertaining, I added neon lighting, music and a virtual audience to congratulate the user on the points they scored.
